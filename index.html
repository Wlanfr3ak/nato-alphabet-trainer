🇬🇧 How to Use

Welcome to the Radio Alphabet Trainer! This tool helps you learn and practice both NATO and German radio alphabets.

  • Multiple Choice Mode:
    • Select the correct radio code word for each letter
    • Instant feedback with color-coded responses
  • Dictation Mode:
    • Listen to code words and type the corresponding letters
    • Practice with names, birth dates, and GPS coordinates
    • Test mode available for formal assessment (10 words)
  • Options:
    • Switch between NATO and German alphabets by clicking score boxes
    • Enable birth dates (DD/MM/YYYY) for additional practice
    • Add GPS coordinates for location training
    • Test mode for accuracy assessment
  • Scoring:
    • Track correct and incorrect answers for each alphabet
    • Percentage success rate for each alphabet
    • Separate test mode scoring with detailed accuracy metrics

🇩🇪 Bedienungsanleitung

Willkommen beim Buchstabiertafel-Trainer! Dieses Tool hilft dir beim Lernen und Üben des NATO- und deutschen Buchstabieralphabets.

  • Multiple-Choice-Modus:
    • Wähle das richtige Buchstabierwort für jeden Buchstaben
    • Sofortiges Feedback mit farblicher Kennzeichnung
  • Diktat-Modus:
    • Höre Buchstabierworte und tippe die entsprechenden Buchstaben
    • Übe mit Namen, Geburtsdaten und GPS-Koordinaten
    • Testmodus für formelle Bewertung (10 Wörter)
  • Optionen:
    • Wechsel zwischen NATO- und deutschem Alphabet durch Klick auf Punkteboxen
    • Aktiviere Geburtsdaten (TT/MM/JJJJ) für zusätzliches Training
    • Füge GPS-Koordinaten für Standorttraining hinzu
    • Testmodus für Genauigkeitsbewertung
  • Punktewertung:
    • Verfolge richtige und falsche Antworten für jedes Alphabet
    • Prozentuale Erfolgsquote für jedes Alphabet
    • Separate Testmodus-Wertung mit detaillierten Genauigkeitsmetriken
